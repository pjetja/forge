<task type="checkpoint:human-action">
  <name>Task 1: Create Figma file and run bootstrap script</name>
  <files>docs/FIGMA-WORKFLOW.md (Figma link update)</files>
  <what-built>
    Claude has prepared docs/figma-bootstrap.js — a Figma Plugin API script that creates all 5 pages and component scaffolding. The user must run this script inside Figma to create the file structure.
  </what-built>
  <how-to-verify>
    **Step 1 — Create the Figma file:**
    1. Open Figma (web or desktop) at figma.com
    2. Go to your personal Drafts (NOT inside a team project — personal Drafts are unlimited on the free plan)
    3. Click "+ New design file"
    4. Rename the file: double-click the title → type "Forge UI Library" → press Enter

    **Step 2 — Run the bootstrap script (Option A: Plugin Console):**
    1. In the Figma file, open the menu → Plugins → Development → Open Console
    2. If "Development" is not visible: go to Plugins → Manage plugins → scroll to bottom → enable "Plugin development"
    3. Open the console, paste the entire contents of `docs/figma-bootstrap.js`
    4. Press Enter / Run
    5. Wait for "Forge UI Library structure created. X components scaffolded." message

    **Step 2 — Alternative (Option B: if console is unavailable):**
    1. In Figma: Plugins → Development → New plugin
    2. When prompted for a template, select "Empty"
    3. In the plugin editor, replace the default code with the contents of `docs/figma-bootstrap.js`
    4. Click Run

    **Step 3 — Verify the structure:**
    Check that Figma now shows 5 pages in the left panel: Tokens, Atoms, Molecules, Organisms, Phase 2: Exercise Library

    **Step 4 — Get the shareable link:**
    1. Click "Share" in the top right
    2. Set permissions to "Anyone with the link can view"
    3. Copy the link

    **Step 5 — Update the docs:**
    Replace `FIGMA_LINK_PLACEHOLDER` in `docs/FIGMA-WORKFLOW.md` with the real Figma link you just copied.
  </how-to-verify>
  <resume-signal>Paste the Figma file link here to confirm it's ready, e.g.: "https://www.figma.com/design/..."</resume-signal>
