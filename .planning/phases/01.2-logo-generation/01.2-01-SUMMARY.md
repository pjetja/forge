## Files Created/Modified

- `src/components/ForgeLogo.tsx` — ForgeLogo React component; variant prop selects horizontal (inline-flex icon+wordmark) or icon-only; className controls sizing
- `src/app/icon.svg` — Raw SVG favicon file; Next.js auto-generates `<link rel="icon">` from this file convention

## Decisions Made

- **Flexbox over dominantBaseline:** First alignment attempt used SVG `dominantBaseline="central"` on the `<text>` element — unreliable in some browsers. Switched to `inline-flex items-center` wrapper with an `h-full` SVG icon beside a `<span>` for the wordmark. More robust and simpler.
- **Hard-coded hex fills:** `#10b981` (emerald) and `#e2e8f0` (slate) are brand constants — intentionally not using `currentColor` or CSS variables so the logo is immune to theme changes.
- **icon.svg in src/app/:** Next.js file-based favicon convention; no `layout.tsx` metadata changes needed.
