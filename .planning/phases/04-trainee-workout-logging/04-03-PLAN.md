```typescript
// Week bounds usage:
import { getCurrentWeekBounds } from '@/lib/utils/week';
const { weekStart, weekEnd } = getCurrentWeekBounds();

// Get trainee's assigned plans:
const { data: assignedPlans } = await supabase
  .from('assigned_plans')
  .select('id, name, status, week_count, workouts_per_week, started_at, created_at')
  .eq('trainee_auth_uid', claims.sub)
  .order('created_at', { ascending: false });

// Get this week's completed session count (for done/remaining):
const { data: weekSessions } = await supabase
  .from('workout_sessions')
  .select('id, assigned_schema_id, assigned_schemas!inner(assigned_plan_id)')
  .eq('trainee_auth_uid', claims.sub)
  .in('status', ['completed'])
  .gte('completed_at', weekStart.toISOString())
  .lte('completed_at', weekEnd.toISOString());

// Check for in-progress session:
const { data: activeSession } = await supabase
  .from('workout_sessions')
  .select('id, assigned_schema_id, assigned_schemas!inner(id, name, assigned_plan_id)')
  .eq('trainee_auth_uid', claims.sub)
  .eq('status', 'in_progress')
  .maybeSingle();
```

Create `src/app/(trainee)/trainee/plans/[assignedPlanId]/page.tsx` as a Server Component.

**Data fetching:**
1. Get Supabase client + getClaims (redirect to /login if missing)
2. Fetch the assigned plan:
   ```
   assigned_plans WHERE id = params.assignedPlanId AND trainee_auth_uid = claims.sub
   ```
   If null → notFound() from 'next/navigation'
3. Fetch all assigned_schemas for this plan:
   ```
   assigned_schemas WHERE assigned_plan_id = params.assignedPlanId ORDER BY slot_index
   ```
4. Fetch this week's completed sessions for this plan (getCurrentWeekBounds):
   ```
   workout_sessions WHERE trainee_auth_uid = claims.sub
     AND status = 'completed'
     AND completed_at >= weekStart AND <= weekEnd
     AND assigned_schema_id IN (schema ids from step 3)
   ```
5. Fetch any in-progress session for this trainee:
   ```
   workout_sessions WHERE trainee_auth_uid = claims.sub AND status = 'in_progress' .maybeSingle()
   ```

Header: Plan name (h1), "{workoutsPerWeek} workouts/week · {weekCount} weeks", back link to /trainee

- Schema name (e.g., "Push Day A")
