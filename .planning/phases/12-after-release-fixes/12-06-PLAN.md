## Task 2 — Add DnD to assigned schema exercise list

**Pattern:** Copy from `src/app/(trainer)/trainer/plans/[planId]/schemas/[schemaId]/page.tsx` DnD setup. The assigned schema editor should use identical `DndContext` + `SortableContext` + drag handle setup.

Steps:

1. Wrap the exercise list in `DndContext` and `SortableContext` from `@dnd-kit`
2. Make each exercise row a sortable item with a drag handle (⠿ icon, same style as plan template)
3. On `onDragEnd`, call a server action `reorderAssignedSchemaExercises(assignedSchemaId, orderedIds)` that updates `sort_order` for each `assigned_schema_exercises` row
4. Add the reorder server action if it doesn't already exist

**Server action signature:**

```ts
export async function reorderAssignedSchemaExercises(
  assignedSchemaId: string,
  orderedIds: string[],
): Promise<void>;
```
