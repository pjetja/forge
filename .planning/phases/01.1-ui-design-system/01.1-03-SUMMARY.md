## Decisions Made

- **Inline SVG for sign-out icon:** lucide-react is not in package.json. Added a minimal inline SVG (door + arrow-right) rather than installing a new library for a single icon. Icon uses `stroke="currentColor"` so it inherits the button's text color and hover states.
- **Next.js Link for logo:** Used `<Link>` from `next/link` (not `<a>`) for the ⚡ Forge logo to get client-side navigation and prefetching behavior consistent with the rest of the app.
- **cursor-pointer scope:** Applied to all interactive button elements across all files touched in this phase. Browsers (especially on macOS) don't always show pointer cursor on `<button>` elements by default; explicit class ensures consistent UX.
