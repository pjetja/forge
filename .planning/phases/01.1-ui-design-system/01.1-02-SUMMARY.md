## Decisions Made
- `or` divider span uses `bg-bg-surface` not `bg-bg-page` — the span sits on the card surface, so it must match the card background to create the "cut through" visual effect
- Google OAuth button uses ghost style (border only, transparent background) — this is the correct secondary action pattern for dark cards where a filled button would compete with the primary action
- Input focus uses `focus:border-accent focus:outline-none` with no ring — provides clean emerald border indication without the blue glow ring artifact
